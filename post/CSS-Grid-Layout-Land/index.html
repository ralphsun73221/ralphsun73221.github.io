CSS Grid 學習資源之一：Layout Land

· 2019-04-20 · # CSS # Grid # Layout

前一陣子在找有關 CSS Grid 的相關學習資源，於是有網友提供了這位設計師的網站以及她的 YouTube，一看之下驚為天人，想說來寫一篇來分享、記錄一下。

Jen Simmons 是 Mozilla 的 Designer 以及 Developer Advocate（這個我不太知道中文應該要怎麼翻，Google 翻譯叫開發者倡導者 ）也曾經在 W3C、Google、Drupal 工作過，有超過 20 年以上的前端設計經驗。

而且她還導過短片，

I also directed several short films which screened at over a hundred film festivals around the globe, including Resfest, Media That Matters, Frameline, International Film Festival Rotterdam, OutFest and NewFest. Called by WFAA TV in Dallas “a brilliantly spliced national address that turns the commander in chief into a dove,” Bush for Peace (2003) played online to more than 150,000 viewers in the era before YouTube. Inclinations (2005), a romantic coming-out comedy, was acquired by MTV Television.

我還導演了幾部短片，這些短片在全球 100 多個電影節上放映，包括 Resfest，Media That Matters，Frameline，鹿特丹國際電影節，OutFest 和 NewFest。 在達拉斯被 WFAA 電視台稱為 “一個精彩的拼接全國演講，將指揮官變成了一隻鴿子”，Bush for Peace（2003）在 YouTube 之前的時代在線播放了超過 150,000 名觀眾。 莫斯科電視台收購了浪漫即將出演的喜劇 “傾向”（2005）。

（請原諒我用 Google 翻譯，但是大家其實也看的懂啦）

她的 YouTube 目前為止已經停止更新，最後一個影片是去年（ 2018 ） 7 月 31 日，而這個頻道雖然叫 Layout Land 但是所有影片都只專注在一個題目上面：關於 CSS 的 Layout 上面。但是最大的篇幅還是在於介紹 Grid。

為何我覺得值得一看

首先就是實在很好懂，只要理解的能力不差，用 Youtube 的自動翻譯翻出來的中文你一定理解意思（我這種英文爛到爆炸我都看得懂了XD），而且有幾集是有熱心的網友翻成中文的。

再來就算真的不懂，每一個影片都有搭配實際的案例，看案例也至少懂七成影片想表達的涵義，不懂的話最好就是跟著做一次，去玩玩看。

看完之後相信我，你會發現新世界 XDDD

不管是在實作還是理論上，裡面的建議基本上屬於基本教義派，也就是比較符合 W3C 的標準規範（這有可能是因為她之前在 W3C 任職過的原因）裡面她常常會拿 W3C 的文件解釋來輔助解釋。

推薦給誰看

我不建議太初學者來看，要看的懂至少需要：

1 基本的 CSS 要懂。
2 懂 Flexbox，甚至你現在已經在用 Flexbox 切版了。

如果有以上基礎的朋友應該會看得很開心，因為每一集都是滿滿的乾糧，沒有廢話，完全都是走使用派的。

後話
